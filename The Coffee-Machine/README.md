# ☕ Coffee Machine Simulator

A Python-based Coffee Machine Simulator that replicates the behavior of a real coffee vending machine. Users can order different coffee drinks, insert coins for payment, receive change, and view machine reports. The system manages resources, processes transactions, and tracks total profit earned.

## 🚀 Features

* Order Espresso, Latte, or Cappuccino
* Resource availability checking
* Coin-based payment system
* Automatic change calculation
* Profit tracking
* Resource consumption management
* Machine status reports
* Modular Object-Oriented Design

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* Classes & Objects
* Dictionaries
* Loops and Conditional Statements

## 📁 Project Structure

```text
Coffee-Machine/
├── main.py
├── coffee_maker.py
├── money_machine.py
├── menu.py
└── README.md
```

### File Descriptions

* **main.py** – Controls the application flow and user interactions.
* **coffee_maker.py** – Handles ingredients, resource checking, and coffee preparation.
* **money_machine.py** – Processes payments, calculates change, and tracks profit.
* **menu.py** – Stores available drinks and their ingredient requirements.

## ⚙️ Available Commands

| Command    | Description                            |
| ---------- | -------------------------------------- |
| espresso   | Order an Espresso                      |
| latte      | Order a Latte                          |
| cappuccino | Order a Cappuccino                     |
| report     | Display available resources and profit |
| off        | Turn off the coffee machine            |

## ☕ How It Works

1. User selects a coffee type.
2. The machine checks available resources.
3. Coins are inserted by the user.
4. Payment is verified.
5. Change is returned (if applicable).
6. Resources are deducted.
7. Coffee is served.
