# 🐍 Classic Snake Game

A classic Snake Game built using Python's Turtle Graphics module. Control the snake, eat food to grow longer, and try to achieve the highest score possible without colliding with the walls or yourself.

## 🚀 Features

* Smooth snake movement using keyboard controls
* Random food spawning
* Score tracking system
* Persistent high score storage using a text file
* Self-collision detection
* Wall collision detection
* Simple and interactive GUI using Turtle Graphics

## 🛠️ Technologies Used

* Python 3
* Turtle Graphics
* Object-Oriented Programming (OOP)

## 📁 Project Structure

```text
Snake-Game/
├── main.py
├── snake.py
├── food.py
├── scoreboard.py
├── data.txt
└── README.md
```

### File Descriptions

* **main.py** – Runs the game loop and handles game logic.
* **snake.py** – Contains the Snake class and movement functionality.
* **food.py** – Creates and manages food spawning.
* **scoreboard.py** – Manages score display and high score tracking.
* **data.txt** – Stores the highest score achieved.

## 🎮 Controls

| Key           | Action     |
| ------------- | ---------- |
| ↑ Up Arrow    | Move Up    |
| ↓ Down Arrow  | Move Down  |
| ← Left Arrow  | Move Left  |
| → Right Arrow | Move Right |

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/CodeXChirag/python-projects.git
```

2. Navigate to the Snake Game folder:

```bash
cd python-projects/Snake-Game
```

3. Run the game:

```bash
python main.py
```
